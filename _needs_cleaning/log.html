Log

Displaying and Exporting Logs

The log menu displays all logs that have been
collected. The logs can be exported to Excel.

1. Application window

The window is divided into four partsparts. The Log Directory Panel is on the left, the Log Panel is in the middle left, and the Reader Panel is on the middle right. On the right of the window are is a toolbar with the Export, Refresh, and Automatic Update Buttons. The orange bar between them can be dragged to hide or show either panel. The Active Log Label above the Reader Panel indicates which log is displayed. The Log Directory Panel is used to select which directory of logs to inspect for files. The files found in the directory specified in the Log Directory Panel will be displayed in the Log Panel ordered from newest to oldest. The file selected in the Log Panel is displayed in the Reader Panel.

If a log file does not exist when data is received, it will be created. The filename is the data of its creation formatted as a 2 digit month, 2 digit day, and 4 digit year separated by underscores (MM_DD_YYYY). 
New log files will also be created at midnight (12:00 AM, 24:00).

Log file data is timestamped and formatted as 2 digit hour, 2 digit minute, 2 digit second, and 3 digit millisecond separated by colons (00:00:00:000)

If a log that is currently being updated as data is received is selected in the Log Panel, the Reader Panel will update in real time.

2. Exporting

Log files can be exported in Excel format (.xlsx). Clicking on the Export Button 
(  ) located above the Log Panel will open a file dialog where the file location and name can be specified.
