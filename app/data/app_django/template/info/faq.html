Are private servers faster?

Yes

What about the public server?

Your analysis will be queued. Analyses are performed in the order they are received. The public server configuration cannot be customized. Leaving an analysis dashboard on the public server will remove all data and settings on the dashboard. Data and settings are only preserved private server.

How much data can I use?

The public server allows up to 0.1 GB of data to be uploaded. There are no storage limits on a private server.
